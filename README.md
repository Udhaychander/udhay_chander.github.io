# Udhay Chander Bharatha — Portfolio

Personal portfolio website built with plain HTML, CSS, and JavaScript. Deployed via GitHub Pages.

🌐 **Live site:** [udhaychander.github.io/udhay_chander.github.io](https://udhaychander.github.io/udhay_chander.github.io)

## About

This is my personal portfolio showcasing my work experience, projects, and technical skills as a Software Engineer based in Chicago, IL. I hold an MS in Computer Science from Illinois Institute of Technology.

---

## Tech Stack

- **HTML5** — semantic structure
- **CSS3** — custom properties, grid, flexbox, animations
- **Vanilla JavaScript** — scroll reveal, nav interactions, counter animations
- **GitHub Pages** — free static hosting with auto-deploy on push

No frameworks. No build tools. No dependencies.

- Responsive design — works on mobile, tablet, and desktop

## Run Locally

No setup required. Just clone and open:

```bash

cd udhay_chander.github.io
open index.html
```
